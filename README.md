# AI Impact on Jobs & Salaries

## Project Overview
This Power BI dashboard analyzes how artificial intelligence is influencing job demand, salary distributions, and employment trends.

## Tools Used
Power BI
Excel

## Key Insights
• Salary levels vary significantly across AI-related roles.
• Certain industries show higher adoption of AI-driven roles.
• Demand for AI skills is increasing globally.
